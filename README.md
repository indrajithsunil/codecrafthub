# CodeCraftHub Learning Platform API

## Overview

CodeCraftHub is a simple learning platform API that allows developers to track courses they want to complete.

This project uses:
- Node.js
- Express.js
- JSON file storage

No database or authentication is used.

---

## Features

- Create courses
- View all courses
- View a single course
- Update courses
- Delete courses
- Course status tracking

Available statuses:

- Not Started
- In Progress
- Completed
